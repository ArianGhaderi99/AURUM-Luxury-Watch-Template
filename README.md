AURUM is a modern luxury watch website template designed to showcase premium products with elegant design, smooth animations and immersive user experience.

The template combines minimalism, performance and luxury aesthetics to create a memorable digital presence for luxury brands.

## 🎨 Customization

### Change Brand Name

Replace:

AURUM

With your own brand name inside HTML files.

---

### Change Main Color

Edit the CSS variable:

--gold: #d4af37;

inside style.css

---

### Replace Images

Replace files inside:

assets/images/

---

### Replace Videos

Replace files inside:

assets/videos/
